## Итерация 4 — Бой: враг + урон + HP

**Цель:** Полная боевая петля Spin → урон врагу → ответный удар → проверка победы/поражения.

- [ ] `src/game/enemies.ts` — данные врагов (HP, атака, паттерны)
- [ ] Компонент `EnemyDisplay` — HP-бар, иконка, имя, статус-эффекты
- [ ] Компонент `CombatLog` — лог событий текущего хода
- [ ] Логика `resolveSymbols` → `player.attack` - `enemy` → `enemy.hp -= damage`
- [ ] Ход врага: `enemy.attack - player.armor` → `player.hp -= effectiveDamage`
- [ ] Броня (`armor`) сбрасывается в начале каждого хода
- [ ] Экран победы/поражения (Game Over)

**Тест:** Spin → числа урона на враге → враг атакует → HP-бары меняются → победа/поражение.
